Identificador

Inicia com @, tendo obrigatoriamente uma letra como segundo caractere e podendo receber letras ou números posteriormente ou finalizar

Ex: @num, @teste123, @i

Palavras reservadas

to - Utilizado no for para delimitar o range da iteração

Variáveis

@num: tipo = valor;
@num = valor;

Valores

Inteiro - Ex: 2, 10, -987
Ponto Flutuante - Ex: 23.9, -98.7, 1.3
String - Ex: "texto", "teste123", "A02ass"
Booleano - True ou False
Null - Null

Operadores

Aritméticos: +, -, *, /
Lógicos: &&, ||, !
Relacionais: ==, !=, <, >, <=, >=


Estrutura if

if (valor operador valor) {
	...
}

Estrutura for

for (@i = 0 to 10) {
	...
}
